Como usar el teclado:

  1. Presione A para cambiar el estado (en sala / ausente)
  2. Ingrese la hora en formato HHMM con 0-9 (ej: 1430 = 14:30)
  3. Para ausencia prolongada (fin de semana, etc.), ingrese 0
  4. Presione * para confirmar y enviar

Otras teclas:

  1. # - Borrar ultimo digito
  2. B - Cancelar y volver sin enviar

Ejemplo:

  1. Al salir hasta las 2pm: A → 1400 → *
  2. Al salir por el fin de semana: A → 0 → *
